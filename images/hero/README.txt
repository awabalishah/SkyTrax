HERO IMAGE FOLDER
==================

Place your main hero/banner image here.

Required File: hero-bg.jpg

Recommended Specifications:
- Dimensions: 1920 x 1080 pixels (minimum)
- Format: JPG or WebP
- File Size: Under 500KB (optimized)
- Subject: Beautiful mosque, Kaaba, or Islamic architecture

Tips:
- Use high-quality images with good lighting
- Ensure there's space in the center for text overlay
- The image will have a dark overlay for text readability
- Consider using images from Unsplash or Pexels

Example file names:
- hero-bg.jpg (main)
- hero-bg-2.jpg (alternative)
- hero-bg-ramadan.jpg (seasonal)
